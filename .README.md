# 🚀 Secure Copilot v2

AI × 音声 × 商談ログを統合する、セールス支援ダッシュボード。

---

## 🎯 概要

`secure_copilot_v2` は、セールス現場の商談・通話・評価・育成を一気通貫で支援するAI管理ダッシュボードです。

- 🧠 GPTによる商談テキスト評価  
- 🎧 音声ファイルからの印象スコアリング  
- 📋 評価スコアログの保存と分析（SQLite）  
- 🛠 管理者向けUIでプロンプト・ユーザー・チームを編集  
- 🏢 PDF資料から会社ビジョンを学習（ナレッジ化）

---

## 📦 機能一覧

| 機能               | 詳細内容 |
|--------------------|----------|
| 🔐 ログイン認証       | ユーザー名とパスワードによるセッション管理 |
| 👤 ユーザー登録       | チーム割り当て・管理者権限付きユーザー作成 |
| 📊 商談評価           | テキスト＋音声アップロードで即時評価 |
| 📝 プロンプト設定     | チーム別にtext/audioプロンプトを編集可能 |
| 🏷 チーム管理         | チーム名・説明・プロンプトキーの編集・削除 |
| 🏢 会社ビジョン学習   | PDFアップロードでナレッジ保存（YAML）

---

## 🖥️ 起動構成

| ファイル | 用途 |
|---------|------|
| `frontend/app.py` | 営業・マネージャー用 商談評価アプリ |
| `frontend/admin_dashboard.py` | 管理者用ダッシュボードUI |

---

## 🔧 セットアップ手順

```bash
# クローン
git clone https://github.com/ict-ryuma/secure_copilot_v2.git
cd secure_copilot_v2

# 仮想環境を作成
python3 -m venv venv
source venv/bin/activate

# 依存パッケージをインストール
pip install -r requirements.txt

# 営業アプリを起動（ポート8501）
streamlit run frontend/app.py

# 管理ダッシュボードを起動（ポート8502）
streamlit run frontend/admin_dashboard.py --server.port 8502
